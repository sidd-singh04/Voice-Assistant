# Voice Assistant Web App

## Description
A React.js web app that responds to user voice commands. Can fetch information from Wikipedia, tell time/date, show maps, provide weather updates, translate words, and more.

## Features
- Voice recognition using Web Speech API.
- Fetches data from Wikipedia and other sources.
- Provides utility functions like jokes, number facts, and translations.
- Responsive and interactive UI.

## Tech Stack
- Frontend: React.js
- APIs: Wikipedia API, Web Speech API
- Tools: VS Code, Git & GitHub
